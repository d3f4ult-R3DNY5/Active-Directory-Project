# Connecting to the PC using the Windows Remote Assistance

## Overview 
---
This step involves connecting to a PC using Windows Remote Assistance. To establish the connection, a Remote Assistance invitation or connection file must be provided and opened to allow access to the system.

### Procedure 
---
1. Go to PC01 ---> `Windows Remote Assistance` --> Click on `Invite Someone you trust to help you` ---> Click on `Save this invitation File`

![](../Attachments/Pasted%20image%2020260210102705.png)

![](../Attachments/Pasted%20image%2020260210102807.png)


![](../Attachments/Pasted%20image%2020260210102824.png)


2. Now On the Domain Controller Go to Remote Desktop Assistance ---> Click on `Help Someone who has invited you` ---> `Use Invitation File`

![](../Attachments/Pasted%20image%2020260210103700.png)

3. Enter the password displayed on **PC01** ---> You will then receive a pop-up similar to the one shown below ---> After this, you can choose to view the screen, request full control, or use the chat feature.
