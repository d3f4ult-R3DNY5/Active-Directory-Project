# Configuring the server 2022

## Overview 
---
This Readme contains the steps of Configuration of the Windows 2022 Server on the VMware workstation

## Procedure
---
1. Once you boot up the Server VM ---> Select the Language, Time and Currency, KB Input ---> install now

![](../Attachments/Pasted%20image%2020260204170420.png)


![](../Attachments/Pasted%20image%2020260204170501.png)

2. Choose the second Option `Standard Evaluation Desktop Experience` ---> next ---> Accept EULA ---> custom

![](../Attachments/Pasted%20image%2020260204170648.png)


![](../Attachments/Pasted%20image%2020260204170748.png)

![](../Attachments/Pasted%20image%2020260204170813.png)


3. Click on Next ---> then The server will install the relevant installation files and automatically restart

![](../Attachments/Pasted%20image%2020260204170921.png)


4. Once it restarts ---> Add the Administrator password for the local account ---> Click on Finish 

![](../Attachments/Pasted%20image%2020260204171407.png)

5. Hit VM ---> Send Ctrl + Alt + Delete  ---> Enter the password of the local admin and login 

![](../Attachments/Pasted%20image%2020260204171543.png)

6. Update the windows server ---> windows settings ---> update and security --- Check for the updates and update that system 

![](../Attachments/Pasted%20image%2020260204172601.png)

7. Restart the system 

![](../Attachments/Pasted%20image%2020260204195119.png)
